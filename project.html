Project Overview

Savory Bistro, a popular local restaurant in Medan, approached me to redesign their outdated website. The existing site was not mobile-friendly, lacked online ordering capabilities, and failed to showcase their culinary offerings effectively. The client needed a modern, responsive website that would enhance their brand image and improve the customer experience.

Project Goals

  • Create a responsive design that works flawlessly across all devices
  • Implement an intuitive online ordering system with secure payment processing
  • Showcase the restaurant's ambiance and menu items with high-quality visuals
  • Improve site performance and loading speeds for better user experience
  • Integrate reservation system with automated confirmation emails

Target Audience

The website was designed with the following user groups in mind:

Local Diners

Regular customers looking for menu updates, special offers, and easy reservation options.

Corporate Clients

Businesses looking for catering services and private event spaces for corporate functions.

Tourists

Visitors to the area searching for authentic local dining experiences with easy-to-access information.

Takeout Customers

Users looking for a quick and efficient online ordering experience for pickup or delivery.

Development Process

1. Research & Discovery

I began with a comprehensive analysis of the restaurant's brand identity, target audience, and competitors. This phase included stakeholder interviews, user surveys, and competitive analysis.

  • Conducted interviews with restaurant staff and management
  • Analyzed 5 competitor websites for best practices
  • Surveyed 50+ existing customers about their online preferences

2. Design & Prototyping

Based on research findings, I created wireframes and interactive prototypes to visualize the user experience. This iterative process involved multiple feedback rounds with the client.

  • Created low-fidelity wireframes for key pages
  • Developed high-fidelity mockups with brand colors
  • Built interactive prototypes for user testing

3. Development & Testing

I implemented the approved designs using modern web technologies, focusing on performance, accessibility, and responsive behavior across devices.

  • Built responsive layouts with HTML5, CSS3, and JavaScript
  • Integrated online ordering system with payment gateway
  • Conducted cross-browser and device testing

Challenges & Solutions

Challenge: Complex Menu Structure

The restaurant's extensive menu with seasonal variations and dietary options was difficult to organize in a user-friendly way.

Solution:

I designed a hierarchical menu system with intuitive filtering options for dietary preferences, price range, and popular items. The menu items were organized by category with clear visual indicators for special items and allergen information.

Challenge: Mobile Performance

Initial designs included high-resolution images that caused slow loading times on mobile devices with limited bandwidth.

Solution:

I implemented responsive image loading with multiple resolution options, lazy loading for off-screen content, and image compression techniques that maintained visual quality while reducing file sizes by over 60%.

Challenge: Integration Complexity

Connecting the online ordering system with the restaurant's existing POS (Point of Sale) system required complex integration work.

Solution:

I developed a custom API middleware that synchronized online orders with the POS system in real-time, while providing fallback mechanisms for manual order processing during system outages.

Lessons Learned

User Testing is Invaluable

Early user testing with actual restaurant customers revealed navigation issues that weren't apparent in initial designs. This led to significant improvements in the final product.

Performance Matters

The initial focus on visual aesthetics led to performance issues. Balancing beautiful imagery with optimization techniques was crucial for mobile users.

Staff Training is Essential

Even the best system requires proper training. Creating comprehensive documentation and training sessions for restaurant staff was vital for successful implementation.
